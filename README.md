✈️ Advanced Airport Management System

A full-stack Airport Management System built using PHP, MySQL, XAMPP, and Bootstrap 5 for a Database Systems coursework project.

This system demonstrates real-world relational database design, normalization, SQL operations, and CRUD-based web development.
